## Algorithm:

### Step 2: Get the input matrix
### Step 3: Find the rank of a matrix
### Step 4: Print the result

## Program:
```python
#Program to find the rank of a matrix.
#Developed by: Prajeeth

import numpy as np
a=np.array([[1,2,3],[3,6,9]])
b=np.linalg.matrix_rank(a)
print(b)
```
